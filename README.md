# Trabalho de REDES 1 para PUC_MG
Trabalho para estudar sockets
## Commands

user nix-shell to get all dependencies
```bash
nix-shell
```

after running nix-shell you can use to recompile both the client and the server
```bash
recompile
```
run server
```bash
./server [port]
```
run client
```bash
./client [ip] [port]
```
